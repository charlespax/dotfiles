# dotfiles
These are my configuration files for various *nix applications (e.g. vim, bash, etc.).

## Usage

### Link Dotfiles
Execute the following command.
```
./link.sh
```

### Setup git
My git preferences are found in `gitsetup.sh`. To configure git execute
the following command.
```
./gitsetup.sh
```
